## 三、核心功能

### 3.1 AI对话系统

**功能特性**:
- ✅ 实时对话交互
- ✅ 历史消息记录
- ✅ 会话管理（清除历史）
- ✅ 优雅的对话UI
- ⏳ 真实AI服务集成（使用模拟数据）

**技术实现**:
- 后端：FastAPI异步接口
- 前端：React Hooks状态管理
- 通信：Axios HTTP请求

### 3.2 工具管理系统

**注册机制**:
- 插件化架构
- 统一元数据格式
- 动态工具加载
- 缓存优化

**已实现工具**:

1. **JSON格式化工具**
   - 格式化JSON（可配置缩进）
   - 压缩JSON
   - 验证JSON格式
   - 统计信息展示
   - 一键复制结果

2. **单位转换工具**
   - 长度转换（米、千米、英里等）
   - 重量转换（千克、克、磅等）
   - 温度转换（摄氏度、华氏度、开尔文）
   - 自动计算结果

3. **进制转换工具**
   - 2/8/10/16进制互转
   - 显示转换公式
   - 输入验证

### 3.3 导航系统

**特性**:
- 树状菜单结构
- 分类展示工具
- 动态加载导航
- 支持折叠/展开

**数据流**:
```
后端工具注册 → 生成导航树 → API返回 → 前端渲染菜单
```

## 六、部署说明

### 6.1 系统要求

- Python 3.8+
- Node.js 16+
- Windows操作系统（或其他支持bash的系统）

### 6.2 快速部署

**方式一：一键启动**
```bash
双击 start.bat
```

**方式二：手动启动**
```bash
# 后端
cd backend
pip install -r requirements.txt
python -m app.main

# 前端
cd frontend
npm install
npm run dev
```

### 6.3 访问地址

| 服务 | 地址 | 说明 |
|------|------|------|
| 前端应用 | http://localhost:3000 | 主应用 |
| 后端API | http://localhost:8000 | API服务 |
